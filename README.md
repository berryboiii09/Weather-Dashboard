# Weather-Dashboard
A Web application built with HTML, CSS and JS that display weather data using OpenWeatherMap API
## Key Features:
Search weather data by city name or fetch data using current location

Dynamic background that adapts to the time of day

Responsive layout optimized for both desktop and mobile

Clean, modern interface focused on usability

Dark Mode toggle with theme persistence via "localStorage"
## Tech Stack
HTML - CSS - JavaScript - OpenWeatherMap API
